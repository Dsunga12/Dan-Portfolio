# Daniel Reyes Sunga Portfolio

A static personal portfolio website built with HTML and CSS. It includes sections for home,
about, skills, work experience, projects, and contact details.

## Files

- `index.html` - page content and smooth-scroll navigation links
- `styles.css` - responsive portfolio styling
- `assets/daniel-sunga-profile.jpeg` - profile photo used in the hero section

## Deploy To Vercel

1. Push this folder to a GitHub repository.
2. In Vercel, choose **New Project** and import the repository.
3. Leave the framework preset as **Other**.
4. Deploy.
